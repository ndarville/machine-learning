[Google Developer Machine Learning Recipes][link]
===========================================

Environment
-----------
* macOS Sierra
* Python 2.7
* Homebrew 1.0.6

Installation
------------
(WIP.)

```sh
brew bundle
pip install -r requirement.txt
```

Running
-------
Run the `script.py` in each video folder:

```sh
python script.py
```
